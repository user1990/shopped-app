# shopped-app
Ecommerce Web application build in NodeJs 

## Features

- NodeJS.
- MongoLab.
- Morgan: loggin request library.
- EJS
- Faker API.
- ElasticSearch.
- Mongoosastic: NodeJs library for ElasticSearch + Mongodb
- Facebook login
- Bcrypt: Hash passwords

## Setup
### Clone Repository
### Run commands
```
npm install
```
```
nodemon
```
### Run in browser
```
Type: http://localhost:3000/
```
