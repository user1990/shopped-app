# shopped-app
Ecommerce Web application build in NodeJs 

## Features

- NodeJS.
- MongoLab.
- Morgan: loggin request library.
- EJS
- Faker API.
- ElasticSearch.
- Mongoosastic: NodeJs library for ElasticSearch + Mongodb
- Facebook login
- Bcrypt: Hash passwords

## Run app
```
1. Clone Repository
```
```
2. npm install
```
```
3. nodemon
```
```
4. type in browser: http://localhost:3000/
```
